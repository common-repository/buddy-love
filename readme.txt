Allows you to show a random sampling of headlines from sites listed in your WordPress blogroll.

== Description ==
[Changelog](http://www.dyers.org/blog/buddy-love-wordpress-widget/#changelog "Buddy Love Change log")

Buddy Love WordPress Widget allows you to display a configurable number of the latest headlines from sites in your WordPress blogroll.  If you have specified a picture in the "Image Address" field for each link in your blogroll, the custom picture will appear to the left of headlines pulled from that site.  The plugin makes use of caching, so it is extremely fast.

Some Features:

* Each site can have its own icon shown beside its headlines.
* The links can be set to "nofollow" if your headlines are coming from paid sources.
* The number of links shown is configurable.
* Buddy caches your feeds to keep things running fast and smooth.

== Installation ==

1. Unzip the entire `buddy-love` directory to your `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go into Presentation/ Widgets and drag the Buddy Love Widget onto your sidebar.
